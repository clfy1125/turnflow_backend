# 다른 DM 툴 충돌 경고 배너 — 프론트 연동 가이드

> 사용자가 같은 인스타그램 계정에 **다른 DM 자동화 툴(매니챗 등)을 함께 연결**해두면 댓글 자동응답이 충돌합니다.
> 이를 예방하기 위해 **연결 완료 시점**과 **대시보드 상단**에 안내 배너를 노출합니다.
> 이 배너는 "안내(정보성)"이며, 사용자가 다른 툴을 실제로 쓰는지 여부는 **자동 감지하지 않습니다.** (감지 한계는 §4 참고)

배너 클릭 시 상세 해제 방법은 → [DISCONNECT_OTHER_DM_TOOLS_GUIDE.md](DISCONNECT_OTHER_DM_TOOLS_GUIDE.md)

---

## 1. 배너 문구 (한국어 카피)

**① 연결 성공 직후 (모달/토스트 내부)**
> 제목: `연결이 완료됐어요 🎉`
> 본문: `매니챗 등 다른 DM 자동화 툴을 같은 계정에 함께 켜두면 댓글 답글이 중복되거나 자동 DM이 발송되지 않을 수 있어요. 다른 툴은 꺼두는 걸 권장해요.`
> 링크(텍스트 버튼): `다른 툴 끄는 방법 보기 →`

**② 대시보드 상단 (닫기 가능한 info 배너, level=warning)**
> `⚠️ 다른 DM 자동화 툴(매니챗 등)이 같은 인스타그램 계정에 함께 연결되어 있으면 댓글 자동응답이 충돌할 수 있어요.`
> 버튼: `해제 방법 보기` / `닫기(×)`

---

## 2. 동작 규칙

| 항목 | 규칙 |
|---|---|
| 노출 대상 | 인스타그램 계정이 **1개 이상 연결된** 워크스페이스 |
| "해제 방법 보기" | [DISCONNECT_OTHER_DM_TOOLS_GUIDE.md](DISCONNECT_OTHER_DM_TOOLS_GUIDE.md) 기반 도움말 페이지/모달로 이동 |
| 닫기(×) | 닫으면 **다시 표시하지 않음** — `localStorage` 또는 사용자 설정에 플래그 저장 (예: `dismissed_dm_conflict_warning:{ig_connection_id}`) |
| 재노출 | 새 인스타그램 계정을 **추가로 연결**하면 그 계정 기준으로 배너 1회 재노출 |
| 연결 성공 배너 | 닫기와 무관하게 **연결 직후 1회**는 항상 표시 |

```js
// 예시: 대시보드 배너 표시 여부
const key = `dismissed_dm_conflict_warning:${igConnectionId}`;
const showBanner = igConnected && !localStorage.getItem(key);

function dismiss() {
  localStorage.setItem(key, "1");
  setShowBanner(false);
}
```

> 배너는 **정보성(항상 노출 후 닫기 가능)** 이 기본입니다. 별도 API 호출 없이 프론트에서 처리합니다.

---

## 3. 접근성 / 스타일

- level: `warning`(노란색 계열), 아이콘 `⚠️`
- 닫기 버튼에 `aria-label="경고 닫기"`
- 링크는 새 창(`target="_blank"`)이 아니라 앱 내 도움말로 이동 권장

---

## 4. (향후·선택) 충돌 "의심" 감지 — 지금은 구현하지 않음

Meta API는 **"이 인스타그램 계정에 어떤 다른 앱이 연결됐는지"를 우리에게 알려주지 않습니다.** 따라서 다른 툴 사용 여부를 확실히 감지할 방법은 없습니다. 다만 아래 **간접 신호**로 "의심"을 띄우는 것은 향후 검토 가능합니다.

- **신호**: 댓글→첫 DM(Private Reply) 발송이 "이미 이 댓글에 답장됨" 류의 사유로 실패 → 다른 툴이 슬롯을 선점했을 가능성.
- **동작(향후)**: 이 실패가 반복되면 대시보드 배너를 `warning` → `error`("다른 툴이 자동 응답을 가로채고 있을 수 있어요")로 승격.
- ⚠️ 어디까지나 **휴리스틱**(앱 특정 불가·공개 답글 미포착)이므로, 이번 배너 범위에는 포함하지 않습니다. 지금은 §2의 정보성 배너만 구현하세요.

---

**요약**: 다른 DM 자동화 툴 충돌은 미리 막는 게 최선이라, **① 연결 직후 1회 + ② 대시보드 상단(닫기 가능)** 에 정보성 경고 배너를 노출합니다. 문구·닫기 규칙은 위 §1~§2를 따르고, "해제 방법 보기"는 [DISCONNECT_OTHER_DM_TOOLS_GUIDE.md](DISCONNECT_OTHER_DM_TOOLS_GUIDE.md)로 연결하세요. 실제 사용 여부 자동 감지는 Meta API 한계로 지금은 하지 않습니다(§4).
